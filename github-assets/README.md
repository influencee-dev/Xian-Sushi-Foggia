# GitHub Assets

In questa cartella puoi caricare i tuoi file multimediali originali per il tuo repository di GitHub.

## File Richiesti:
- **logo.png**: Il logo trasparente del ristorante (quello originale che hai inserito). Sovrascrivi il file `public/logo.png` con la tua versione originale per vederla applicata correttamente in tutto il sito.
- **immagini dei piatti**: Sostituisci i file in `/src/assets/images/` con le tue immagini originali scontornate o a sfondo nero per un effetto ottimale.
